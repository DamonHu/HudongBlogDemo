# 通过cocos2d-x3.10简单手势识别的一个小游戏

这个游戏就是通过手势消除障碍物的，很简单的游戏，只是为了示例下上篇写的《cocos2d-x3.10简单手势识别》的一个具体demo。

## 具体功能

通过不同手势消除不同障碍物。

左右躲避障碍物

添加手势轨迹

分数计算
